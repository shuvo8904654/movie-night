# Movie Night

A retro Windows 98-themed random movie generator web app.

Built with Flask and the TMDB API. Uses 98.css for the classic desktop look.

## Features
- Random movie discovery from TMDB's popular movies
- Windows 98 retro UI with starfield background
- Responsive mobile layout with full-screen windows
- About page styled as a Notepad readme

## Setup
```
pip install -r requirements.txt
python app.py
```

## Tech Stack
- Python / Flask
- TMDB API
- 98.css
